# Contribuer à CyCode

Merci de votre intérêt pour **CyCode** ! Nous sommes ravis d'accueillir de nouveaux contributeurs. Pour garantir la qualité et la fiabilité de cet assistant, veuillez suivre ces quelques règles.

## 🚀 Comment contribuer

1. **Signaler un bug** : Si vous rencontrez un problème, ouvrez une [Issue](https://github.com/Simonc44/cycode/issues). Soyez précis et joignez une capture d'écran si nécessaire.
2. **Proposer une amélioration** : Vous avez une idée pour rendre l'UI/UX encore plus incroyable ? Ouvrez une discussion ou une issue pour en débattre.
3. **Soumettre du code** :
    - Forkez le dépôt.
    - Créez une branche dédiée : `git checkout -b feature/nom-de-la-fonctionnalite`.
    - Faites vos modifications.
    - Soumettez une **Pull Request (PR)** vers la branche `main`.
